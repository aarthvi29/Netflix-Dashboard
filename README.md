**1. Project Title**

**🎬Netflix Analytics Hub:** Interactive Content & User Analytics Dashboard

An interactive Power BI dashboard designed to analyze Netflix's content library and user engagement through dynamic visualizations, KPI cards, and business intelligence techniques.

**2. Purpose**

The Netflix Analytics Hub is an end-to-end Power BI solution that combines **Content Analytics** and **User Analytics** into a single interactive platform. The dashboard helps analyze content trends, user behavior, subscription insights, and business performance using data-driven visualizations and DAX measures.

**3. Tech Stack**

The dashboard was built using the following technologies:

* 📊 **Power BI Desktop** – Dashboard development and visualization.
* ⚙️ **Power Query** – Data cleaning, transformation, and preprocessing.
* 🧮 **DAX (Data Analysis Expressions)** – KPI calculations and custom measures.
* 🔗 **Data Modeling** – Relationships between multiple datasets.
* 📈 **Interactive Visualizations** – KPI Cards, Donut Charts, Bar Charts, Treemaps, Maps, and Line Charts.
* 🎨 **Custom UI Design** – Netflix-inspired dashboard theme with page navigation.
* 📁 **File Format** – `.pbix` for development and `.png` for dashboard previews.

**4. Data Source**

The project uses publicly available Netflix datasets containing information related to content and user behavior.

The datasets include:

* Movies and TV Shows
* Genres
* Ratings
* Directors
* Countries
* Release Years
* User Profiles
* Subscription Plans
* Watch Hours
* Device Usage
* Payment Methods
* Customer Demographics

The datasets were transformed and modeled to create meaningful business insights through interactive reporting.

**5. Features**

🎯 **Business Problem**
Streaming platforms need to understand both **content performance** and **user engagement** to improve recommendations, optimize content investment, and enhance customer experience.

🎯 **Dashboard Goal**

Develop an interactive Business Intelligence solution that enables users to explore Netflix content performance and user behavior using dynamic reports and KPI-driven insights.
